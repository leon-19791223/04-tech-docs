# B-4：配置管理增强

## 1. 现状分析

| 问题 | 位置 | 影响 |
|:-----|:------|:------|
| 无参数校验 | `app.py` 直接取前端的值用 | 无效 IP/端口导致服务器 500 错误 |
| 环境预设硬编码 | `core_engine.py` 只有嘉兴银行 | 扩展其他银行需改 vendor 代码 |
| 错误提示不友好 | 服务器抛 Exception | 用户看到英文堆栈 |

## 2. 改动文件清单

| 文件 | 操作 | 内容 |
|:-----|:------|:------|
| `engine/config_validator.py` | **新增** | 参数校验规则引擎 |
| `conf/environment_presets.json` | 已有(B-0) | 多银行预设存储 |
| `app.py` | **修改** | 接入参数校验 |
| `templates/dws_config.html` | **修改** | 前端错误提示 |

## 3. 懒人友好设计

- 输入即校验：输完立刻提示
- 错误字段红色边框 + 中文纠正提示
- 预设模板一键填充
