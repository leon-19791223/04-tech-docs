# DWS 部署工具集 · 测试方案与报告

> **基线**: `dws-iter-prep` (基于 `dws-ui-v3`)  
> **测试范围**: 全部 13 个页面 + API + 交互功能  
> **测试类型**: 功能测试 / 渲染测试 / 回归测试

---

## 测试框架

每个迭代步骤包含以下测试层级：

### 层级 1：代码级验证（自动）
```
python -m py_compile app.py                    → 语法检查
python -c "from app import app"                → 模块导入
```

### 层级 2：渲染验证（自动，test_client）
```
GET / → 200 + 内容包含关键元素
GET /api/health → 200 + status=ok
```

### 层级 3：功能验证（自动）
```
场景切换 → override 生效
凭据管理 → store/get/expire 正确
配置校验 → 正确/错误配置分别校验
```

### 层级 4：视觉检查（手动）
```
浏览器打开页面，检查布局/颜色/字体/间距
```
