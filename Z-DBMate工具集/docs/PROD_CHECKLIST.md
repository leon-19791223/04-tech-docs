## 部署命令

确认全部勾选后，执行：

```bash
# root模式
bash scripts/deploy.sh -m root -r "密码" -s "单节点TPOPS+三节点GaussDB"

# service模式
bash scripts/deploy.sh -m service -p "密码" -s "单节点TPOPS+三节点GaussDB"

# 或交互式（手动输入密码）
bash scripts/deploy.sh -s "单节点TPOPS+三节点GaussDB"
```

监控进度：

```bash
bash scripts/monitor.sh --tail
```
