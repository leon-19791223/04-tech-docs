# HCCDE-GaussDB 考试练习系统 — 使用说明

> **版本**: v1.0.0 · **题库**: 147 题 / 6 章 / 3 类型  
> **适用**: 华为 HCCDE-GaussDB 认证考试

---

## 一、产品概述

HCCDE-GaussDB 考试练习系统是专为华为 HCCDE-GaussDB 认证考试设计的练习平台。内置 **147 道真题**，覆盖全部 6 个章节，支持章节练习、随机组卷、模拟考试三种模式，并附带错题本与成绩统计功能。

### 功能模块

| 功能 | 说明 |
|------|------|
| 章节练习 | 按章节逐章练习，每章独立进度 |
| 随机组卷 | 从全部题库中随机抽取指定数量的题目 |
| 模拟考试 | 限时 90 分钟，模拟真实考试环境 |
| 错题本 | 自动收录错题，支持重做和清空 |
| 成绩统计 | 按章节维度展示正确率，掌握度一目了然 |
| 快捷背题 | 单页显示所有题目及答案，快速浏览复习 |
| AI 智能分析 | 调用大模型解析题目难点（需联网） |

## 二、安装与启动

### 方式一：绿色版（推荐）

1. 将 `HCCDE-GaussDB.exe` 复制到任意目录
2. **双击运行**即可
3. 无需安装、不写注册表

### 方式二：安装版

1. 双击 `HCCDE-GaussDB_Setup_v1.0.0.exe`
2. 选择安装目录（默认 `C:\Program Files\HCCDE-GaussDB`）
3. 安装完成后自动运行
4. 桌面和开始菜单会生成快捷方式
5. 可在控制面板卸载

## 三、使用指南

### 3.1 首页

启动后进入主界面，左侧为功能导航区，右侧为主要操作区。

- **章节选择**：单击章节名称进入对应练习
- **进度条**：显示每章已完成题数 / 总题数
- **正确率**：每章的答题正确率百分比

### 3.2 章节练习

1. 点击任意章节进入练习模式
2. 系统**逐题显示**，答完一题自动进入下一题
3. 答错自动加入错题本
4. 右上角显示当前进度（如 3/25）

### 3.3 随机组卷

1. 点击「随机组卷」
2. 设置题目数量（默认 20 题）
3. 系统从全部题库中**随机抽取**
4. 支持「上一题/下一题」导航
5. 提交后显示得分和解析

### 3.4 模拟考试

1. 点击「模拟考试」
2. **90 分钟倒计时**自动开始
3. 系统随机抽取 60 题（覆盖所有章节）
4. 可随时点击「交卷」提前结束
5. 交卷后显示成绩、用时、每道题答案与解析

### 3.5 错题本

- 自动记录所有答错的题目
- 支持单题重做和全部重做
- 可清空错题本（已掌握后）

### 3.6 成绩统计

- 按章节显示：做题数、正确数、正确率
- 绿色进度条直观展示掌握程度
- 帮助定位薄弱章节

### 3.7 快捷背题

- 单页展示某章节所有题目及答案
- 适合考前快速过一遍知识点
- 支持章节切换

### 3.8 AI 智能分析

- 选中任意题目，点击「AI 分析」
- 调用大模型解析题目难点和考点
- **需要联网**，首次使用会提示配置 API

## 五、数据文件说明

用户数据存储在程序所在目录的 `data/user_data.json`，包含：

- 错题本记录
- 各章节练习成绩
- 模拟考试历史

> **备份建议**：定期备份 `data/user_data.json`，重装或升级后恢复即可保留学习记录。绿色版请直接复制整个 `data` 文件夹。

## 六、常见问题

### Q1: 启动后显示空白/闪退？
- 确保系统为 **Windows 10/11**
- 尝试以管理员身份运行

### Q2: 错题本数据丢失？
- `data/user_data.json` 与 exe 在同一目录
- 如被杀毒软件清理，重新做题即可生成

### Q3: AI 分析无法使用？
- 需要网络连接
- 首次使用需在设置中配置 API Key

### Q4: 如何更新题库？
- 替换 `hccde_quiz.py` 文件，重新打包
- 或联系开发者获取新版 exe
